# 🎬 Movies Database Analysis using SQL

## 📌 Project Overview
This project is a SQL-based analysis of a movies dataset.  
The main goal is to explore movie data and extract useful insights like ratings, genres, actors, and year-wise trends using SQL queries.

---

## 🗂️ Database Details

### Database Name:
movies_db

### Table Name:
movies_data

### Table Schema:
- movie_id (INT, Primary Key)
- title (VARCHAR)
- year (INT)
- genre (TEXT)
- rating (FLOAT)
- story (TEXT)
- actor (TEXT)

---

## 🛠️ Tools Used
- MySQL
- SQL (Structured Query Language)

---

## 📊 SQL Operations Performed

### 🔍 Data Exploration
- View all movie records
- Count total movies in dataset
- Find unique genres
- Find oldest and newest movie years

---

### 🎭 Filtering Data
- Filter movies by genre (Action)
- Search movies by actor (Aamir Khan)
- Get movies released after 2009
- Search movies using story keywords (LIKE operator)

---

### ⭐ Rating Analysis
- Top 10 highest rated movies
- Average rating of all movies
- Movies with rating greater than 8

---

### 📈 Aggregation Analysis
- Number of movies per genre
- Number of movies per actor

---

### 🎯 Advanced Filtering
- High-rated movies of specific actors
- Movies containing “love” in story

---

## 📌 Key Insights
- Action is one of the most common genres
- Several movies have high ratings above 8
- Actor-based filtering helps identify popular actors
- Year-wise filtering shows modern movie trends
- Aggregation helps understand genre popularity

---

## 💡 SQL Concepts Used
- SELECT statements
- WHERE clause
- LIKE operator
- ORDER BY
- GROUP BY
- COUNT() and AVG()
- Aggregation functions
- Filtering and sorting data
